# Project_BILLY

Obstacle Detection using YoloV3. 

Built from AlexeyAB's <a href="https://github.com/AlexeyAB/darknet" title="Darknet">Darknet</a> and Validated by <a href="https://github.com/MiguelARD/DoorDetect-Dataset" title="DoorDetect-Dataset">DoorDetect-Dataset</a>. Custom data set created by downloading images from google images and using <a href="https://github.com/heartexlabs/labelImg" title="labelImg">labelImg</a>. to create labels used for training.
